# Welcome to Inkwell

A beautiful, minimal Markdown viewer for macOS.

## Features

- **Clean Design**: Notion-inspired typography and spacing
- **Syntax Highlighting**: Beautiful code blocks with highlight.js
- **GitHub Flavored Markdown**: Tables, task lists, and more
- **Native macOS Feel**: Seamless file associations

## Code Examples

Here's some JavaScript:

```javascript
function greet(name) {
  return `Hello, ${name}!`;
}

const message = greet('World');
console.log(message);
```

And some Python:

```python
def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n - 1) + fibonacci(n - 2)

print([fibonacci(i) for i in range(10)])
```
